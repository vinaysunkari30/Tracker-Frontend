# Tracker Frontend

A React-based project tracking and cost management application that helps users organize projects, track expenses, and manage project-related data through an intuitive user interface.

## Features

- Create and manage projects
- Add, edit, and delete project items
- Track project expenses
- Responsive user interface
- Real-time updates
- Authentication support
- Backend API integration

## Tech Stack

### Frontend
- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap

## Installation

### Clone Repository

```bash
git clone https://github.com/vinaysunkari30/Tracker-Frontend.git
```

### Navigate to Project Directory

```bash
cd Tracker-Frontend
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will run at:

```text
http://localhost:3000
```
